# 우리의 회고

친구들과 매주 `이번 주 회고`와 `다음 주 계획`을 작성하고, 지난 회고를 아카이브로 확인하는 GitHub Pages용 정적 웹앱입니다.

## 구성

- `PRODUCT_PLAN.md`: 이미지 분석 기반 페이지 및 기능 기획
- `index.html`: 이번 주, 지난 회고, 상세 화면 구조
- `design-system.css`: 공통 토큰과 기본 요소
- `styles.css`: 스크린샷 기반 화면 레이아웃
- `app.js`: 사용자 선택, 자동 저장, 저장, 아카이브 상세 전환 로직

## 주요 기능

- 이번 주 회고 홈
- 참여 상태 칩
- 선택한 멤버의 지난주 계획 표시
- 누구의 회고든 선택해서 이번 주 회고와 다음 주 계획 작성
- 전체 멤버 회고 카드 표시 및 카드별 편집 진입
- 지난 회고 아카이브 목록
- 주차별 회고 상세 화면

## 저장 방식

별도 서버나 빌드 단계 없이 동작합니다. 현재 주차의 입력값과 사용자 선택은 브라우저 `localStorage`에 저장됩니다.

## 실행

정적 파일만으로 구성되어 있어 바로 열 수 있습니다. 로컬 확인이 필요하면 다음 명령을 사용합니다.

```sh
python3 -m http.server 8000
```
